  • Product Metafield

    In default, Shopify not allow you add new custom field for product. So with Basel we will help you understand how to bring custom field to your store.

    Example: you can have different description, different video... for each product.

    The first you need install this extension on your chrome by go to this link: SHOPIFYFD

    After install you will see the icon in your browser:

    When go to Edit Product, you need click to ShopifyFD icon and metafield will show.

    You need enter fill these sections: Namespace, Key, Value, then click Save.

    Basel Theme support metafield for: Countdown, Video, Short Description, External/Affiliate, Tag, Username and Custom Tab.

    Countdown:

    If want show countdown on page single product, you need enter value date time for it.

    1. namespace: meta

    2. key: countdown

    3. value: value (yyyy/mm/dd)

    Result:

    Video:

    If want show video, you need enter youtube video link for it.

    1. namespace: meta

    2. key: video

    3. value: youtube url

    Result:

    Short Description:

    Allow you show different short description for each product instead cut off full description.

    1. namespace: meta

    2. key: description_excerpt

    3. value: description excerpt content

    Result:

    External/Affiliate:

    Allow you enter external/Affiliate link, Add cart function will be hidden.

    1. namespace: external

    2. key: text on button

    3. value: link to external product

    Result:

    Tag:

    Allow show your tag on instagram.

    1. namespace: meta

    2. key: hashtag

    3. value: #(tag on instagram)

    Result:

    Username:

    Allow show user on instagram.

    1. namespace: meta

    2. key: username

    3. value: username on instagram

    Result:

    Background:

    Allow show background color on product single.

    How to get photos from the library:

    1. namespace: meta

    2. key: background_image

    3. value: paste background image

    Result:

    Custom Tab:

    If want add custom tab for single product, you need enter name and content for it.

    1. namespace: additions

    2. key: Tab Name

    3. value: Tab Content

    Result:

    - Per page allowed to show an instagram. Priority to display the tags.

    - Sometime you don't see the metafield on product page or it said not loaded, please reload and click to metafield icon on browser again.
